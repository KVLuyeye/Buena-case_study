# Buena Case-Study
  A quick case-study that mocks a shortened login process of Buena.

## Installation
  Clone the repository:
    git clone https://github.com/KVLuyeye/Buena-case_study.git

  Install dependencies:
    npm install

  Run application: 
    npm run dev
  
  Run test:
    npm test

  Run tests in watch mode:
    npm run test:watch
